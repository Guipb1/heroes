# Hero Factory

Plataforma de gestão de heróis: criação, listagem, edição, ativação/desativação e exclusão, com busca e paginação.

## 🧱 Stack

- **Frontend:** Next.js + TypeScript + Tailwind CSS
- **Backend:** Node.js + Express + TypeScript
- **Banco de dados:** MySQL 8 (via Docker)
- **ORM:** Prisma
- **Validação:** Zod
- **Testes:** Jest + Supertest

## 📦 Pré-requisitos

- [Node.js](https://nodejs.org/) 18 ou superior
- [Docker](https://www.docker.com/) (para rodar o MySQL)

## 🚀 Como rodar o projeto

### 1. Banco de dados (MySQL no Docker)

Na raiz do projeto:

```bash
docker compose up -d
```

Isso sobe um MySQL 8 em `localhost:3306` (banco `hero_factory`).

_(As instruções de backend e frontend serão adicionadas conforme o projeto avança.)_
